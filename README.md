# 📈 AI-Powered Stock Price Prediction and Recommendation System

## Overview

This project is an AI-driven stock market analysis platform that predicts future stock prices using Machine Learning and Deep Learning techniques. The system combines Random Forest, LSTM Neural Networks, and News Sentiment Analysis to provide intelligent Buy, Hold, and Sell recommendations.

The project also calculates Confidence Scores and Risk Levels to help users understand the reliability of predictions.

---

## Features

✅ Historical Stock Price Prediction

✅ Random Forest Regression Model

✅ LSTM Deep Learning Model

✅ Ensemble Prediction (RF + LSTM)

✅ News Sentiment Analysis

✅ Confidence Score Calculation

✅ Risk Level Assessment

✅ Buy / Hold / Sell Recommendation Engine

✅ Interactive Flask Dashboard

## Machine Learning Models

### Random Forest Regressor

Used for:
- Learning relationships between stock features and closing prices
- Fast and accurate predictions

Performance:
- MAE: 0.423
- MSE: 0.911
- R² Score: 0.9997

### LSTM Neural Network

Used for:
- Capturing time-series patterns
- Learning long-term stock price dependencies

### Ensemble Model

Final prediction is generated by combining:

Final Prediction = (RF Prediction + LSTM Prediction) / 2

This improves overall prediction stability.
