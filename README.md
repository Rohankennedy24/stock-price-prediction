# AI Powered Stock Price Prediction System

## Technologies
- Python
- Flask
- Random Forest
- LSTM
- Sentiment Analysis
- Chart.js

## Features
- Stock Price Prediction
- News Sentiment Analysis
- Buy/Hold/Sell Recommendation
- Confidence Score
- Risk Analysis
- Interactive Dashboard

## Dataset
Dataset Name: NVIDIA Historical Stock Dataset

Source: Kaggle

Records: 2626

Columns:
1. Date
2. Open
3. High
4. Low
5. Close
6. Adj Close
7. Volume

Downloaded from Kaggle and used for training Random Forest and LSTM models.
